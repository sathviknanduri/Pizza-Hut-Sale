# Pizza Hut Sales Analysis 🍕📊

## Project Overview
This project provides a comprehensive sales analysis for Pizza Hut to identify patterns in consumer behavior, product performance, and revenue drivers.

## 🔑 Key Insights
* **Top Products:** Classic and Chicken categories consistently generate the highest sales.
* **Size Impact:** Medium and Large pizzas contribute significantly more to total revenue than Small sizes.
* **Peak Trends:** Sales peak during weekends and standard meal hours (lunch/dinner).
* **Consumer Behavior:** High frequency of multi-pizza orders, proving the effectiveness of bundled deals.

## 🛠️ Tools Used
* **Data Analysis:** SQL / Python (Pandas)
* **Visualization:** Power BI / Tableau (or Excel)
* **Documentation:** Microsoft Word/PDF
